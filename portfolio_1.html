⚙️ 設定 Google Sheets 報價來源

這是目前最穩定的台股報價方法，利用 Google Sheets 的 GOOGLEFINANCE() 函式，完全免費、不需要 API key。

📋 設定步驟
① 確認 Apps Script 的 prices 分頁已有資料（執行 updatePrices 一次）
② 右上角「共用」→「一般存取權」→「知道連結的任何人」→「檢視者」→ 完成
③ （重要）「檔案 → 共用 → 發佈到網路」→ 選 prices 分頁 → 「發佈」
④ 從試算表網址複製 ID（/d/ 和 /edit 之間那段）貼到下方
你的持倉對應代號：
取得方式：試算表上方 → 檔案 → 共用 → 發佈到網路 → 選 prices 分頁 → 格式選「逗號分隔值(.csv)」→ 發佈 → 複製連結

✏️ 手動輸入現價

若 API 無法連線（休市、防火牆等），可在此手動輸入各股現價。留空則沿用均成本估算。
